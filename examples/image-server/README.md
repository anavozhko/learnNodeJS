# Image-server

This is an express server for images saving and providing.

#### Requirements:

- node.js version > 8
- npm

#### Set up

1. `cd image-server/`

2. Install dependencies:

   `npm install`

3. Setup env variables:

   `cp env.example .env`

   Replace variables at .env by what you need

4. Update config `/config`, if you need

#### Running

Run application:

`npm start`
